# 💱 Conversor de Moedas

🔗 **[Acesse o projeto online aqui](https://naiurysamily.github.io/Conversor-de-moedas/)**

## 💻 Sobre o Projeto
Este é um conversor de moedas dinâmico desenvolvido para colocar em prática os fundamentos do Front-end e a integração com serviços externos. O projeto permite realizar a conversão entre diferentes moedas utilizando taxas de câmbio precisas e atualizadas.

## ⚙️ Funcionalidades
* Conversão de valores entre diversas moedas globais.
* Consumo da **ExchangeRate-API** para obter cotações em tempo real.
* Interface interativa com atualização imediata dos resultados na tela.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação da página.
* **CSS3:** Estilização e layout.
* **JavaScript:** Lógica da aplicação, manipulação do DOM e requisições assíncronas.
* **ExchangeRate-API:** API RESTful externa utilizada para fornecer os dados de câmbio.

## 📚 Principais Aprendizados
* Consumo de APIs RESTful utilizando o método `fetch` no JavaScript.
* Manipulação e extração de dados no formato **JSON**.
* Trabalho com código assíncrono (Promises / Async & Await) para lidar com a resposta da API sem travar a tela.
* Captura de eventos e atualização dinâmica da interface (DOM) com base nos dados recebidos do servidor.
